# 📊 Sales Performance Analysis using SQL

## 🚀 Project Overview
This project analyzes sales performance, customer behavior, and product insights using a Retail_Sales database.

It demonstrates SQL skills using:
- Joins  
- Aggregations  
- Subqueries  
- Window Functions  

---

## 🎯 Objective
- Explore relational datasets  
- Perform data analysis using SQL  
- Generate actionable insights  

---

## 🗂️ Dataset
Tables used:
Customer, Employee, Product, Category, Supplier, SalesOrder, OrderDetail, Shipper

---

## 🛠️ Skills
- SQL (MySQL)
- Joins & Aggregations
- Subqueries
- Window Functions

---

## 📈 Key Insights
- Top categories generate highest revenue  
- High-value customers drive sales  
- Sales trends vary over time
